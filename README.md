# BasicPrologFunctions
Code in Prolog done to show knowledge in the language. Functions include defining basic familial relations, finding the max element in a list, checking if two lists have any elements in common, checking if a list is the union of two others, and if an element is the last one in a list.

Note that this was an academic assignment. That means that the function headers as well as what program should do were given by the professor and to be carried out by students, or me.
Code can be run on SWISH: https://swish.swi-prolog.org/
